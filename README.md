# Taiwan Bankruptcy Risk Modeling

Machine learning benchmark study for predicting corporate financial distress using the Taiwanese Bankruptcy Dataset.

## Overview
This repository investigates bankruptcy prediction as a binary classification problem using statistical learning and ensemble methods.

## Models Evaluated
- Logistic Regression
- Random Forest
- XGBoost
- Support Vector Machine

## Pipeline
1. Data preprocessing
2. Feature engineering
3. Class imbalance handling
4. Model training
5. Evaluation and explainability
